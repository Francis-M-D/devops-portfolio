## 🚀 Quick Deployment (3 Steps)

### Step 1: Create GitHub Repository
```bash
# Go to github.com/new
# Create repository: "devops-portfolio"
# Clone it
git clone https://github.com/YOUR_USERNAME/devops-portfolio.git
cd devops-portfolio
```

### Step 2: Copy Files
```bash
# Copy all .html and .md files to the repository folder
# Your folder structure should look like:
# devops-portfolio/
# ├── index.html
# ├── architecture.html
# ├── project-deep-dives.html
# ├── README.md
# └── DEPLOYMENT_GUIDE.md
```

### Step 3: Push & Deploy
```bash
git add .
git commit -m "Initial commit: DevOps portfolio"
git push -u origin main

# Then enable GitHub Pages in Settings > Pages
# Select "Deploy from a branch" > main > root
```

**Your portfolio will be live at:** `https://YOUR_USERNAME.github.io/devops-portfolio`
